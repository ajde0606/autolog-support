Privacy

Autolog is a privacy-first app that runs entirely on your device. We collect no user data and have no server infrastructure.

Summary: Autolog processes all data locally on your iOS device. No data is ever transmitted to any servers. Optional features that require external services are clearly labeled and disabled by default.

Data We Access

  • Microphone Access: Required for recording voice notes
  • Speech Recognition: Used to transcribe your voice to text
  • Local Storage: Used to store your notes and categories
  • Optional Gemini API: If you provide your own API key, we can use Google's Gemini model for improved classification

Data We Do NOT Collect

  • No account creation
  • No personal identifiers
  • No analytics
  • No third-party tracking
  • No server communication

Classification Options

  • Local Classification (Default): Uses rule-based keyword matching, entirely offline
  • Gemini Classification (Optional): Uses Google's Gemini API with your own API key. All requests are sent directly to Google; we do not proxy or store any data.

Data Security

  • All data is stored locally on your device
  • No remote backups or cloud sync
  • Removing the app will delete all stored data
  • iOS backup (if enabled) may include your Autolog data

Permissions

Autolog requests the following permissions:

  • Microphone: to record audio
  • Speech Recognition: to convert speech to text
  • Notifications: for task reminders

We request the minimal set of permissions required for the app to function. You can manage these permissions in iOS Settings.

Data Retention

All data remains on your device. If you delete the app, all data will be removed according to iOS data handling policies.

Frequently Asked Questions

How does classification work?

Autolog uses two classification methods:

  • Local (Default): Rule-based classification using keyword matching - completely offline and private
  • Gemini API (Optional): AI-powered classification using Google's Gemini - requires your own API key

What are the available categories?

Autolog supports the following fixed categories:

  • Work
  • Study
  • Family
  • Social
  • Food & Drink
  • Fitness
  • Music
  • Game
  • Book
  • Medical
  • Shopping
  • Sports
  • Travel
  • Weather
  • Finance

How do I get a Gemini API key?

  1. Visit Google AI Studio
  2. Sign in with your Google account
  3. Create a new API key
  4. Copy the key and paste it into Autolog Settings

Note: This is completely optional. Autolog works perfectly without Gemini API.

Can I edit the classification?

Yes! Tap any note to view details, where you can:

  • Change the category
  • Toggle between to-do and done
  • Edit the transcript text
  • Delete the note

Does Autolog work offline?

Yes! Autolog is designed to work completely offline using local classification. The Gemini API feature is optional and only used when enabled and internet is available.

Is my data backed up?

Your data is stored locally on your device. If you have iCloud or iTunes backup enabled for your device, your Autolog data will be included in those backups.

What languages are supported for transcription?

Autolog uses Apple's speech recognition, which supports multiple languages. The available languages depend on your iOS version and device settings.

Troubleshooting

  • Microphone not working: Check that you've granted microphone permissions in iOS Settings > Autolog
  • Speech recognition not working: Ensure you've granted speech recognition permissions
  • Gemini not classifying: Verify your API key is entered correctly and you have internet connection
  • App crashes: Try restarting the app or your device

隐私说明

Autolog 是一款完全在设备端运行的隐私优先应用。我们不收集任何用户数据，也没有服务器基础设施。

概要： 所有数据均在你的 iOS 设备本地处理，从不传输到任何服务器。需要外部服务的可选功能会清晰标注，且默认关闭。

我们会访问的数据

  • 麦克风权限： 用于录制语音笔记
  • 语音识别： 将语音转换为文字
  • 本地存储： 保存笔记与分类
  • 可选 Gemini API： 仅在你提供 API Key 时，用于更智能的分类

我们不会收集的数据

  • 无需注册账号
  • 不收集个人身份信息
  • 无统计分析
  • 无第三方追踪
  • 无服务器通信

分类方式

  • 本地分类（默认）： 基于关键词匹配的规则引擎，完全离线
  • Gemini 分类（可选）： 使用你提供的 Google Gemini API Key，所有请求直连 Google，我们不代理、不存储

数据安全

  • 全部数据保存在设备本地
  • 无远程备份或云同步
  • 卸载应用即删除所有数据
  • 若启用 iOS 备份，备份中可能包含你的 Autolog 数据

权限说明

Autolog 需要以下权限：

  • 麦克风：用于录音
  • 语音识别：用于转写
  • 通知：用于任务提醒

我们仅请求维持功能所需的最小权限，你可在 iOS 设置中随时管理。

数据保留

所有数据都留在你的设备上。删除应用后，数据会按 iOS 规则被移除。

AutoLog 提供两种分类方式：

  • 本地（默认）： 基于关键词规则的离线分类，完全私密
  • Gemini（可选）： 依赖你提供的 Gemini API Key，使用 Google 的模型进行分类

如何获取 Gemini API Key？

  1. 访问 Google AI Studio
  2. 使用 Google 账号登录
  3. 创建新的 API Key
  4. 复制后粘贴到 Autolog 设置中

提示： 该功能完全可选，不影响核心使用。

可以手动调整分类吗？

可以！打开任意笔记即可：

  • 修改分类
  • 切换待办或完成状态
  • 编辑转写文本
  • 删除笔记

离线可用吗？

可以。默认的本地分类完全离线。Gemini 功能需要网络且仅在启用时使用。

数据会备份吗？

数据保存在本地。如果设备启用了 iCloud 或 iTunes 备份，Autolog 数据会包含在系统备份中。

转写支持哪些语言？

AutoLog 使用苹果的语音识别，支持多语言，具体取决于 iOS 版本和设备设置。

故障排查

  • 麦克风不可用： 检查在 iOS 设置 > Autolog 中是否授权麦克风
  • 语音识别不可用： 确认已授予语音识别权限
  • Gemini 未分类： 核对 API Key 是否正确并确认网络连接
  • 应用崩溃： 尝试重启应用或设备
